\begin{abstract}
\noindent
[STUB — to be written when experiment is executed.]
This paper tests whether mathematical constants ($e$, $\pi$, $i$, $\sqrt{2}$) are encoded in LLM weights primarily as numerical values (metric) or as associations with process types (topology). The manifesto predicts topology is deeper than metric: the process-association ($e \leftrightarrow$ exponential growth) should be more thermally robust than the numerical value ($e \approx 2.71828$). Thermal spectroscopy across temperatures $T \in \{0.0, 0.3, 0.7, 1.0, 1.5, 2.0\}$ tests this prediction.
\end{abstract}
